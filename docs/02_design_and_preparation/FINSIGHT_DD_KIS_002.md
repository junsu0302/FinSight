---

## 1. 문서 개요

본 문서는 FinSight 프로젝트의 **ETL** 단계의 결과로, **KIS(한국투자증권) API(기본시세)**를 통해 수집되어 Data Lake(`data_lake`)에 저장되는 원천 데이터의 구조와 속성을 정의합니다.

## 2. 작성 원칙

본 문서의 모든 내용은 아래의 작성 원칙을 준수합니다.

1.  **완전성**: KIS API로부터 수집되는 모든 원천 데이터 테이블과 컬럼은 반드시 정의되어야 합니다.
2.  **명확성**: 컬럼명과 설명은 FinSight 프로젝트의 컨텍스트를 모르는 사람도 이해할 수 있도록 구체적으로 작성합니다.
3.  **일관성**: 유사한 의미를 가진 데이터는 프로젝트 전반에 걸쳐 일관된 컬럼명과 데이터 타입을 사용합니다.
4.  **현행성**: API 명세 변경 등 데이터 구조에 변경이 발생할 경우, 본 문서는 즉시 업데이트되어야 합니다.

## 3. 테이블 명세

KIS API의 시세 관련 엔드포인트를 통해 수집된 각 테이블의 명세를 기술합니다.
